Beside thousands of game ideas noted here and there, I actually have two working prototypes.
One is a puzzle game I am calling colors, and the other is a reflex based game called series.
Below is more detail about them, including live demos with detailed implementation bits.
Both prototypes were made using Phaser.

Colors

Idea Sprout: May 2014
First Prototype: May 2014, Python / PyGame
Web Prototype: April 2017, HTML5 / JS / Phaser

Rules:

  • Cell values: 1-6, Cell Neighbors: Left, Right, Top, Bottom
  • Neighbor has (own value + 1) -> Cell can be incremented short press & release
  • 1 -> 2 -> 3 -> 4 -> 5 -> 6 -> 1
  • Cell cluster with same value can all change at once. long press & release on a changable cell
  • CHANGING CLUSTERS COUNTS FOR A SINGLE MOVE
  • Goal: Turn all cells into the same color in least amount of moves.

Series

Idea Sprout: Summer 2015
First Prototype: Summer 2015, AppGameKit2
Web Prototype: Autumn 2016, HTML5 / JS / Phaser

Rules:

  • 9 Numbers appear on the screen.
  • Press the numbers in the right order.
  • Gain time for each correct press.
  • More numbers appear.
  • Goal: Press numbers in correct order as fast as you can, without timeout.
Design & Code: Erinç Argımak (mail), 2017. Tested on Windows 10/Chrome, 1920x1080. External Font: Patua One
